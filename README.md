# NewsReader App 📰

NewsReader merupakan aplikasi Android sederhana yang dikembangkan menggunakan Jetpack Compose. Aplikasi ini menampilkan daftar berita terbaru dari API secara real-time dengan tampilan modern dan responsif.

## Fitur

* Menampilkan berita terbaru (Top Headlines)
* Mengambil data dari API (NewsAPI)
* Menampilkan judul, deskripsi, dan gambar berita
* Tampilan modern dengan Jetpack Compose
* Arsitektur sederhana (API → ViewModel → UI)
* Error handling (jika gagal mengambil data)
* Loading data secara asynchronous
